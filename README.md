# react-music-player

***

项目使用yeman脚手架，进行react webpack项目环境搭建。音乐的数据源用的是阿里云上免费的QQ排行音乐查询接口。

使用的技术包括react-router、pubsub 、bootsrap 、jQuery 、jPlayer、localStorage等。

因为使用的了bootsrap 所以在样式上兼容PC和移动。
